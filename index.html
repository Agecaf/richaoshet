How to play

Use W, A, S, D together with Q, E to control the spaceship. Press Space to shoot.

By default there's some nice automatic drag, but if you get hit by asteroids, or your own bullets, you'll have to control the spaceship manually. If the spaceship goes too fast, it'll be destroyed! So be sure to not lose control. Press O to restart.

Settings

Use the following box to change the following settings: The character's controls. Then press the button to evaluate. Is this a lazy answer? Could UI have been made for this? Yes! But this is better than nothing, and easy to implement.
